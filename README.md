# Roman Numerals Code Challenge
### Objective:

Use Test Driven Development to create a tool that converts any integer value into its roman numeral equivalent. 


### Basic Roman Numerals:

| Roman Numeral | Arabic Numeral |
|------|----------|
| I | 1 |
| V | 5 |
| X | 10 |
| L | 50 |
| C | 100 |
| D | 500 |
| M | 1000 |


### Examples:

| Arabic | Roman |
|------|----------|
| 4 | IV |
| 9 | IX |
| 14 | XIV |
| 44 | XLIV |
| 944 | CMXLIV |
